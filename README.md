# Muhabet

Duolingo clone built using:
<ul>
  <li>Next.js</li>
  <li>React</li>
  <li>Drizzle</li>
  <li>PostgreSQL</li>
  <li>Clerk</li>
  <li>Stripe</li>
  <li>Vercel</li>
</ul>

Allows users to learn Albanian, a language not yet available on Duolingo.
