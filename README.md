# Muhabet

This project is a Duolingo clone being built using:
<ul>
  <li>Next.js</li>
  <li>React</li>
  <li>Drizzle</li>
  <li>PostgreSQL</li>
  <li>Clerk</li>
  <li>Stripe</li>
</ul>

It will allow users to learn Albanian, a language not yet available on the popular language learning app.
